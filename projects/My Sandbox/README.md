# My Sandbox

A workspace for testing random ideas and experiments with Cursor.

## Structure

- `experiments/` - Test projects and experimental code
- `scripts/` - Standalone utility scripts
- `docs/` - Documentation and notes
- `data/` - Data files and datasets
- `templates/` - Reusable templates and boilerplate code

## Usage

Feel free to organize your experiments however makes sense. This structure is just a starting point!
